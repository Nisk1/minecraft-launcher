## About
A Python CLI offline launcher for Minecraft that supports multiple accounts, offline skins, and seamless switching between all available versions.

## Features
- CLI interface.
- Supports multiple accounts.
- Offline skin support.
- Automatic resourcepack handling.
- Easy switching between installed Minecraft versions.
- Supports additional options such as disabling multiplayer or chat.
